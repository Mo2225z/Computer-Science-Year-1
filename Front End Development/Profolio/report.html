Planning

  • Accessibility aspect 1

    • All the images on my website have an alt tag which ensures those with visual impairments will not be put at a disadvantage.

  • Accessibility aspect 2

    • I've put an lang attribute on each page which is used to identify the language of text content on the web pages. Screen readers use this to switch language profiles to provide the correct accent and pronunciation.

  • Accessibility aspect 3

    • I've set the font sizes to appropriate sizes and I've used consistent semantic tags throughout the 3 web pages for screen readers to be able to better read the site.

How have you made the site easy to navigate?

The navigation bar appears at the same place no matter the webpage and I've set it so once you hover over the links in the navigation bar the colour changes to red using CSS. This works really well because it highlights important links that can be clicked on for users
